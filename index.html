About Train Time

Train Time is a mobile-friendly train schedule application that incorporates Firebase and Moment.js to host arrival and departure data.

This project has helped me learn how to use Google's Firebase Realtime Database and Moment.js together, creating a useful tool that is interactive and inspiring.
